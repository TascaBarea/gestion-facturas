## 🐛 BUG CRÍTICO CORREGIDO

### Problema: Filtro de subcadenas

```python
# ANTES (buggy):
if any(x in descripcion.upper() for x in ['CANT', ...]):
# 'CANT' in 'PICANTE' = True → ¡Filtraba productos válidos!

# DESPUÉS (corregido):
palabras = descripcion.upper().split()
if any(x in palabras for x in ['CANT', ...]):
```

**Impacto:** FERRIOL 1T25 pasó de DESCUADRE a OK (+1 factura)

**Archivos corregidos:**
- `embutidos_ferriol.py`
- `alcampo.py`
- `hernandez.py`
