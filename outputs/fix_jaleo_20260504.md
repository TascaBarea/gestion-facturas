## Extractor

`Parseo/extractores/jaleo.py` (117 líneas). Decorador
`@registrar('ACEITES DE ESPECIALIDAD JALEO', 'ACEITES JALEO', 'JALEO')`,
clase `ExtractorJaleo(ExtractorBase)`.

Atributos:
- `nombre = 'ACEITES DE ESPECIALIDAD JALEO'`
- `cif = 'B75298836'`
- `iban = ''` (TJ — definido en MAESTRO)
- `metodo_pdf = 'pdfplumber'`
- `categoria_fija = 'ACEITE'`

Métodos:
- `extraer_total`: 2 patrones (moderno + presupuesto). Cuidado especial con
  `^TOTAL\s*:` para no capturar `TOTAL PARCIAL`.
- `extraer_fecha`: `Fecha dd/mm/yyyy` o `dd-mm-yyyy` → canonical `dd/mm/yyyy`.
- `extraer_referencia`: `Factura N\d+` (prioridad sobre `#\d+`, porque PDF1
  contiene `Observaciones #1859 ENVÍO URGENTE 24H` que NO es la REF).
- `extraer_lineas`: TODO `→ []`. Las tablas varían entre formatos, no
  bloquean el rellenado de Excels.
- `extraer_forma_pago`: NO implementado. MAESTRO ya define `FORMA_PAGO=TJ`.
