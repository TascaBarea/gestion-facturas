## FUNCIONALIDADES IMPLEMENTADAS

### 1. Modo interactivo (`main.py`)

```
python main.py    # Sin argumentos

============================================================
PARSEAR FACTURAS - SELECCIÓN DE CARPETA
============================================================

Opciones:

  E. 📁 Explorar con ratón (abrir selector de carpetas)
  M. Introducir ruta manualmente
  Q. Salir
```

- **Opción E**: Abre diálogo nativo de Windows para seleccionar carpeta
- **Opción M**: Permite escribir ruta manualmente
- Si detecta carpetas de trimestres, las muestra numeradas
- Pregunta si procesar solo ATRASADAS cuando detecta la subcarpeta

### 2. Campo # mejorado (`parser.py`)

| Tipo | Detección | Formato salida |
|------|-----------|----------------|
| Normal | 4 dígitos al inicio | `4006` |
| Atrasada | 3 dígitos + palabra ATRASADA | `460 ATRASADA` |
| Atrasada | 3 dígitos en carpeta ATRASADAS | `946 ATRASADA` |
| TEMP | Empieza con código trimestre (4T25...) | `TEMP001_4T25` |
| Error | 3 dígitos sin ATRASADA fuera de carpeta | `ERROR_3DIG_946` |
| Ignorar | Empieza con Z, NO ES FACTURA | `0` |

**Contador TEMP por trimestre:** Reinicia en cada ejecución, independiente por trimestre.

### 3. Soporte ATRASADAS (`main.py`)

- Búsqueda automática de subcarpeta `ATRASADAS`
- Procesa archivos de carpeta principal + ATRASADAS juntos
- Opción de procesar solo ATRASADAS por separado
- Archivos marcados con `[ATR]` en el log de procesamiento

### 4. Hoja Facturas mejorada (`excel.py`)

Nuevo orden de columnas:
```
#, ARCHIVO, CUENTA, TITULO, Fec.Fac., REF, TOTAL FACTURA, Total, OBSERVACIONES
```

| Columna nueva | Descripción |
|---------------|-------------|
| ARCHIVO | Nombre del archivo PDF (sin extensión) |
| TOTAL FACTURA | Total extraído del PDF por el extractor |
| Total | Total calculado (suma de líneas: base × IVA) |

Esto permite ver fácilmente si hay descuadre comparando TOTAL FACTURA vs Total.

## REGLAS DOCUMENTADAS

### Campo # - Reglas de numeración

| Serie | Rango | Uso |
|-------|-------|-----|
| Normal | 1001-9999 | Facturas del trimestre actual (4 dígitos) |
| Atrasadas | 900-999 | Facturas de trimestres anteriores (prioritario) |
| Atrasadas | 400-499 | Si se agota el rango 900 (secundario) |

### Carpeta ATRASADAS

- Nombre exacto: `ATRASADAS` (mayúsculas)
- Solo un nivel de profundidad
- Contiene facturas de trimestres anteriores contabilizadas en el actual
- El # es único por año (nunca colisionan normales con atrasadas)

### IVA especial 2024

- En 2024 existió IVA 2% para alimentos básicos
- En 2025 volvió al 4%
- Si factura atrasada es de 2024 (ej: `488 ATRASADA 4T24...`), IVA 2% es válido

## PROBLEMAS RESUELTOS

### 1. Corrupción en Excel consolidado
**Problema:** Columna # mostraba decimales (1156.680059) en consolidación manual.
**Causa:** Fórmulas de Excel copiadas incorrectamente.
**Solución:** Usar pandas concat para consolidación limpia.

### 2. Archivos sin número de gestoría
**Problema:** 176 archivos en 4T25 sin número asignado (formato `4T25 MMDD...`).
**Solución:** Sistema TEMP que asigna `TEMP001_4T25`, `TEMP002_4T25`, etc.

### 3. Detección de ATRASADAS
**Problema:** Archivos de 3 dígitos no se detectaban como atrasadas.
**Solución:** Parser detecta 3 dígitos + palabra ATRASADA o carpeta ATRASADAS.

### 4. Ruta de Dropbox no encontrada
**Problema:** Menú interactivo no encontraba carpetas de trimestres.
**Solución:** Añadir opción E para selector gráfico con ratón.

## APRENDIZAJES TÉCNICOS

### Selector de carpetas con tkinter
```python
import tkinter as tk
from tkinter import filedialog

root = tk.Tk()
root.withdraw()
carpeta = filedialog.askdirectory(title="Selecciona carpeta")
```

### Detección de 3 dígitos vs 4 dígitos
- Regex `^\d{3}\s+` para 3 dígitos
- Regex `^\d{4}\s+` para 4 dígitos
- Orden importante: primero detectar 3, luego 4

### Contador TEMP por trimestre
```python
_CONTADOR_TEMP_POR_TRIMESTRE = {}  # {'4T25': 3, '1T26': 1}

def _obtener_siguiente_temp(trimestre):
    _CONTADOR_TEMP_POR_TRIMESTRE[trimestre] += 1
    return f"TEMP{contador:03d}_{trimestre}"
```
