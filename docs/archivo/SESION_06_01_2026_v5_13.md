## EXTRACTORES CORREGIDOS

### 1. MARITA COSTA (10/10 ✓)

**Problema:** IVA incorrecto - asignaba todo al 10% cuando algunos productos llevan 4% o 2%.

**Causa raíz:** 
- AOVE → IVA 4% (o 2% en 2024 por reducción temporal)
- LUCÍA PICOS → IVA 10% hasta mayo 2025, luego 4%
- Resto productos → IVA 10%

**Solución:** Asignación por combinatoria usando desglose fiscal de la factura:
```python
def _asignar_iva_por_desglose(self, lineas, desglose):
    # 1. Identifica IVA reducido en factura (2% o 4%)
    # 2. Encuentra candidatos (AOVE, LUCÍA PICOS)
    # 3. Prueba todas las combinaciones
    # 4. Selecciona la que minimiza diferencia con desglose
```

**Cambios adicionales:**
- Unión de líneas multilínea (productos con descripción en 2 líneas)
- Soporte IVA 2%/4%/10%

---

### 2. ANGEL Y LOLI (4/4 ✓)

**Problema:** Portes no prorrateados + patrón rígido que no capturaba "CUENCO 10 CM".

**Causa raíz:**
- El cuadro fiscal tiene columnas: TIPO IMPORTE DESCUENTO PORTES BASE IVA
- Si no hay DESCUENTO, los valores se desplazan
- El patrón regex no permitía números en la descripción

**Solución:** Detección automática PORTES vs DESCUENTO:
```python
# Si IMPORTE + val2 ≈ BASE → val2 es PORTES
# Si IMPORTE - val2 ≈ BASE → val2 es DESCUENTO
```

**Cambios adicionales:**
- Patrón flexible: `r'^(.+?)\s+(\d+)\s+(\d+[.,]\d{2})\s+(\d+[.,]\d{2})$'`
- Prorrateo proporcional de portes/descuentos
