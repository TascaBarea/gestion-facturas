### 5.7.6 Reglas de Negocio

**PORTES/ENVГҚO:** SIEMPRE distribuir proporcionalmente entre productos. NUNCA lГӯnea separada.

**IVA:** Usar tipo real del producto (4% lГЎcteos/pan, 10% alimentaciГіn, 21% servicios/bebidas alcohГіlicas). Cada extractor define el IVA de sus productos.

### 7.3 FunciГіn extraГӯda: buscar_factura_candidata() (v1.5)

LГіgica comГәn a transferencias, compra_tarjeta y adeudo_recibo (~40 lГӯneas cada uno вҶ’ 1 funciГіn):

```
1. Buscar facturas por importe (Вұ0.01вӮ¬)
2. Calcular fuzzy scores con aliases (calcular_similitud_con_aliases)
3. Filtrar por umbral вүҘ70%
4. Filtrar facturas ya usadas (facturas_usadas global)
5. Desempatar por fecha: factura anterior mГЎs cercana вүӨ60 dГӯas
6. Fallback: mejor fuzzy si no hay fecha vГЎlida
```

**ParГЎmetros que diferencian los 3 clasificadores:**

| ParГЎmetro | transferencia | compra_tarjeta | adeudo_recibo |
|---|---|---|---|
| `fecha_operativa` fallback | SГӯ | SГӯ | No (solo fecha_valor) |
| `incluir_ref` (campo Factura) | No | No | SГӯ |
| `log_candidatas` | SГӯ | No | SГӯ |

### 7.4 OptimizaciГіn buscar_mejor_alias() (v1.5)

**Antes (v1.4):** `df_fuzzy.iterrows()` por cada movimiento вҶ’ 565 filas Г— 3945 mov = 2.2M comparaciones pandas

**DespuГ©s (v1.5):** `construir_indice_aliases()` genera `Dict[str, str]` una sola vez:
- Match exacto: `dict.get()` вҶ’ O(1) en lugar de filtro DataFrame
- Match fuzzy: itera `dict.items()` вҶ’ sin overhead pandas (~10x mГЎs rГЎpido)
